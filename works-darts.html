サポート終了したダーツマシンの再生

メーカーサポート終了により補修部品の入手が困難となった業務用ダーツマシンに対し、 制御システムを独自開発基板へ置き換えることで、既存筐体を活かした再生を実現しました。

概要

業務用ダーツマシンの多くは、専用ソフトウェアを搭載したPCやAndroid端末と、 盤面判定・LED装飾・コイン投入などを扱う専用I/O回路によって構成されています。

本事例では、サポート終了により修理・補修が困難となった既存筐体に対して、 Raspberry Pi Compute Module 4を搭載した専用制御基板を新規設計し、 制御系を全面的に置き換えることで、再び運用できる状態へ再生しました。

課題

ダーツというゲーム自体は大きく変わらない一方で、メーカーによるサポート終了後は 補修部品の入手が困難となり、故障した店舗は高額な新台へ入れ替えるか、 使えないまま放置するしかない状況にありました。

筐体や盤面などの機械部分はまだ使用できるにもかかわらず、 電子制御部品の故障によって機器全体が使えなくなる点が大きな課題でした。

提案

既存筐体をそのまま活かし、内部の制御システムのみを独自開発基板へ置き換える構成を提案しました。 これにより、店舗側は筐体を買い替えることなく、安価にダーツマシンを再稼働できるようになります。

当初は故障の多い周辺部品の互換機提供から開始し、段階的に開発範囲を拡大。 最終的には、オリジナルのダーツゲームを搭載したメイン制御部まで提供可能な構成としました。

開発内容

制御基板には Raspberry Pi Compute Module 4 を採用し、 盤面判定回路、LED装飾制御、コイン投入制御など、 ダーツマシンに必要な専用I/O機能を1枚の基板に統合しました。

回路設計・基板設計を自社で行い、製造委託先で製造された基板を最終検査したうえで提供。 ソフトウェア面では、組込みLinux上で動作する制御ソフトウェアと、 店舗利用を想定したオリジナルダーツゲームを開発しました。

また、現地で不具合が発生した場合に迅速に修正できるよう、オンラインアップデート機能も搭載しました。

成果

本システムは、業務用ダーツマシン向け基板提供として約3年間継続して提供され、 卸先を通じて全国約500店舗規模に導入されました。

店舗側は既存筐体を活かしながら、制御系のみを置き換えることで、 サポート終了機器を再び営業利用できる状態にすることができました。

担当範囲

本事例では、単なる基板設計やソフトウェア開発に留まらず、 既存機器の動作解析から、専用制御基板の設計、組込みソフトウェア、 ゲームソフトウェア、製造後検査、アップデート対応まで一貫して担当しました。

  • 既存機器の動作解析
  • 専用制御基板の回路設計・基板設計
  • Raspberry Pi Compute Module 4を用いた組込みLinux環境構築
  • 盤面判定・LED装飾・コイン投入などのI/O制御
  • オリジナルダーツゲームの開発
  • オンラインアップデート機能の実装
  • 製造委託後の最終検査
  • 小ロット量産・継続提供

この事例で示せること

本事例は、単にダーツマシンを開発した事例ではありません。 メーカーサポートが終了した業務用機器を解析し、専用ハードウェアとソフトウェアによって 再生・延命した事例です。

古い設備を活かしたい、メーカー補修ができなくなった機器を再生したい、 市販品では実現できない専用制御システムを作りたい、といったご相談にも応用できます。
